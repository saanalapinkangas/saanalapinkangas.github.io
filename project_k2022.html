PROJEKTI: Pankkiautomaatin simulaatio

Ensimmäisen vuosikurssin sovellusprojekti K2022
Tieto- ja viestintätekniikka, Oulun Ammattikorkeakoulu

Projektissa suunnitellaan ja toteutetaan pankkiautomaattisimulaattori, jolla voidaan simuloida pankkiautomaatin perustoimintoja: rahan nosto, saldon ja tilipahtumien tarkasteleminen. Kohdejärjestelmän arkkitehtuuri on asiakas-palvelin (client-server).

Projektissa käytetään RFID-kortteja ja RFID-kortinlukijaa simuloimaan pankki-/luotto-/kaksoiskortteja. Kolmen hengen ryhmätyö.

Oma vastuualueeni projektissa oli RFID-lukijan käyttöönotto, RestApin login, Postmanin käyttöönotto, ikkunoiden ajastimet, uloskirjautumistoiminnot ja ulkoasu.


Toteutustapa: mm. C++, JavaScript, Postman, Node.js, RestAPI, RFID-kortti ja -kortinlukija
GitHub: löydät projektin koodit täältä

PROJECT: Bank ATM Simulation

First-Year application project S2022
Bachelor of Engineering, Information Technology, Oulu University of Applied Sciences

The project involves designing and implementing a bank ATM simulator that can simulate basic ATM functions: cash withdrawal, checking balances, and viewing transactions. The target system architecture is client-server.

In the project, RFID-cards and an RFID-cardreader are used to simulate bank/credit/debit cards. A team project involving three members.

My responsibility in the project included setting up the RFID reader, implementing the RestAPI login, using Postman, managing window timers, implementing logout functionality, and designing the user interface.


Implementation: C++, JavaScript, Postman, Node.js, RestAPI, RFID card and card reader
